# Install

Plugin:

```shell
asdf plugin add jq https://github.com/jblaker/asdf-jq.git
```

jq:

```shell
# Now jq commands are available
jq --help
```

Check [asdf](https://github.com/asdf-vm/asdf) readme for more instructions on how to
install & manage versions.
